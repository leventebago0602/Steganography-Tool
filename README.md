Steganography Tool - Image Text Hiding in Python
Ez a projekt egy Python-alapú szteganográfiai eszköz, amely lehetővé teszi titkos üzenetek elrejtését képekben, és azok visszafejtését numerikus kulcs használatával. A program a pixelértékek kismértékű módosításával tárolja az információkat, a változás szabad szemmel alig észrevehető.

Könyvtárak:
OpenCV (cv2): Képkezelés és pixelműveletek.
NumPy: Tömbműveletek a képadatok feldolgozása.
Pillow (PIL): Képek megnyitása és mentése.
Matplotlib: Eredmények megjelenítése.
